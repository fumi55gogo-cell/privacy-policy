# プライバシーポリシー

  筋トレトラッカー（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、個人情
  報の保護に努めます。

  ## 収集する情報

  本アプリは以下の情報を収集します：
  - Googleアカウント情報（認証のため）
  - ワークアウト記録データ（Firebase Firestoreに保存）
  - 広告配信のためのデバイス情報（Google AdMob使用）

  ## 情報の利用目的

  収集した情報は以下の目的で使用します：
  - アプリ機能の提供・改善
  - ユーザーデータのクラウド同期
  - 広告の表示

  ## 第三者へのデータ提供

  以下のサービスにデータが共有される場合があります：
  - Google Firebase（データ保存・認証）
  - Google AdMob（広告配信）

  これらのサービスのプライバシーポリシーはGoogleのプライバシーポリシーに準じます
  。
